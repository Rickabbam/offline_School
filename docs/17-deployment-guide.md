# Deployment Guide

## Environments
- Local development
- Test/staging
- Production

## Deployment modes
### Online mode
- Standard connected install
- Delta updates with checksum verification

### Offline/low-connectivity mode
- USB-distributed installer
- Offline patch package installation
- Deferred sync after first stable connection

## Update strategy
- Online differential patches where possible
- Offline patch bundles for schools with unstable internet
