# Roles and Permissions

## Core roles
- `admin`
- `cashier`
- `teacher`
- `parent`
- `student`
- `support_technician`
- `support_admin`

## Support roles
### support_technician
- Installation, setup, printer configuration, diagnostics, backup/restore assistance
- No finance visibility or sensitive student access by default

### support_admin
- SaaS/platform troubleshooting and controlled impersonation
- All impersonation must be audited

## Campus-scoped permissions
- Most operational actions are campus-scoped
- Cross-campus access requires explicit role grants and online verification
