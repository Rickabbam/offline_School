# MOBILE APP SPEC

## Role
Scoped teacher and parent experience.

## Teacher workflows
- attendance capture
- score entry where approved
- timetable / class assignment lookup
- notifications

## Parent workflows
- fee/invoice visibility
- payment history / receipts where enabled
- attendance visibility
- report card / result visibility
- notifications

## Mobile principles
- narrow subset of desktop capabilities
- offline only where explicitly justified
- sync rules compatible with backend and desktop
- no unsafe local authority for sensitive finance/admin workflows
