# PRODUCT SPEC — FULL SYSTEM

## Product
offline_School is an offline-first school management platform for Ghanaian schools.

## Product surfaces
- Windows desktop app: primary campus operations client
- Mobile app: teacher and parent workflows
- Web admin: school management, reporting, admin configuration
- SaaS admin: platform operations, tenant support, controlled impersonation
- Backend API: source of truth for cloud persistence, sync, auth, reporting, and cross-campus consistency

## Core domains
- Identity and access
- Onboarding / school setup
- Academic configuration
- Admissions
- Students and guardians
- Staff and teaching assignments
- Attendance
- Exams / assessments / results / report cards
- Finance lifecycle
- Reports and exports
- Backup / restore / reconciliation
- Sync center / conflict resolution / diagnostics

## Ghana-specific needs
- receipt and fee workflows aligned to school practice
- school types and starter templates
- low-connectivity operation
- Windows-first device reality
- external backup/export support
