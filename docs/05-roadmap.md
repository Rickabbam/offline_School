# Roadmap

## Phase A — Foundation
Monorepo, NestJS backend shell, Flutter desktop shell, SQLite local DB, sync queue, installer baseline.

## Phase B — Operational MVP
School setup wizard, admissions, students, staff, attendance.

## Phase C — Finance + Retention
Fees, posting/reversal lifecycle, receipts, reminders, arrears.

## Phase D — Exams + Mobility
Exams/results/report cards, teacher and parent mobile workflows.

## Phase E — SaaS Maturity
Tenant management, billing, support console, analytics, cross-campus reporting.
