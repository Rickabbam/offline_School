# Problem Statement

Schools in low-connectivity environments face frequent internet outages, unstable power, and low-end hardware constraints.

## Current pain points
- Web-only tools become unusable during outages
- Finance and attendance operations are interrupted
- Data entry is delayed and error-prone
- Backups and recovery are inconsistent

## Product response
A local-first platform with SQLite on device and cloud sync to PostgreSQL when connectivity is available.
