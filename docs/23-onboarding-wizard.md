# Onboarding Wizard

## Goal
Allow a school to become operational in 30–45 minutes on first run.

## First-run steps
1. School profile
2. Campus setup
3. Academic year and terms
4. Classes, arms, subjects
5. Grading scheme
6. Staff roles
7. Fee categories and structures
8. Receipt format
9. Notification settings
10. Device registration
11. First admin confirmation

## UX notes
- Use guided validation and defaults
- Provide starter templates for common Ghana school types
